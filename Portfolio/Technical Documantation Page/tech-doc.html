HTML Fundamentals

HTML is the backbone of web development and defines the structure of web pages.

It uses tags to create elements like headings, paragraphs, and links.

<html> <head> <title>My Page</title> </head> <body> <h1>Welcome</h1> </body> </html>
  • Semantic HTML improves accessibility
  • Tags are case-insensitive
CSS Styling

CSS controls the presentation of web pages, including colors and layouts.

It can be applied inline, internally, or externally via stylesheets.

body { font-family: Arial, sans-serif; color: #333; }
  • CSS selectors target elements
JavaScript Basics

JavaScript adds interactivity to web pages, enabling dynamic content.

It runs in the browser and manipulates the DOM.

let message = "Hello, World!"; console.log(message);
  • Variables: let, const, var
  • Functions enhance reusability
Responsive Design

Responsive design ensures websites work well on all devices.

Media queries and flexible layouts are key techniques.

@media (max-width: 600px) { body { font-size: 16px; } }
Web APIs

Web APIs allow interaction with browser features and external services.

Examples include the DOM API and Fetch API for HTTP requests.

fetch('https://api.example.com') .then(response => response.json()) .then(data => console.log(data));
